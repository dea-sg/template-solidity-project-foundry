# setup

## install foundry

```
// for mac or linux
curl -L https://foundry.paradigm.xyz | bash;
foundryup
```

```
// for windows
cargo install --git https://github.com/foundry-rs/foundry --bins --locked
```

```
// for docker
docker pull ghcr.io/foundry-rs/foundry:latest
```

## install library
```
yarn
```

## build and test

```
forge build
forge test
```
